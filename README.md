# Billing SaaS Platform

A production-grade billing SaaS built with Laravel and React.

## Features
- Client management
- Invoice creation & lifecycle
- PDF invoice generation
- Email notifications
- Stripe payments
- Recurring invoices
- Activity tracking
- Multi-tenant architecture

## Tech Stack
- Laravel 12
- React + Inertia
- MySQL
- Stripe
- Railway deployment

## Architecture Highlights
- Service-based backend
- Async email & PDF processing
- Secure Stripe webhooks
- Clean SaaS lifecycle handling

## Why I Built This
To demonstrate real-world SaaS architecture beyond CRUD apps.
